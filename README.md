## 🚀 快速开始

### 使用 Skill Manager

```bash
# 列出所有 skills
python3 .skill-manager/skill-manager.py list

# 搜索 skills
python3 .skill-manager/skill-manager.py search <关键词>

# 同步到 GitHub
python3 .skill-manager/skill-manager.py sync
```

### 直接使用

Hermes Agent 会自动加载 `~/.hermes/skills/` 下的所有 skills。
